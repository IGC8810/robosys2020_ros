# robosys2020_ros

## 概要
ロボットシステム学の課題。ROSでパッケージの作成を行った。<br>


## 環境
* OS : ubuntu20.04
* ROS : noiteic

## 実行コマンド
$ roscore<br>
$ rosrun robosys2020_ros publish.py <br>
$ rosrun robosys2020_ros subscribe.py <br>
